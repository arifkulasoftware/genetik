# genetik
Genetik Algoritma Kullanılarak En verimli yolu bulmak için Görselleştirilmiş uygulama

1) Genetic.exe çalıştır
2) mouse ile ekranda noktalar tıklanarak belirle
3) Popülasyon sayısı Örneğin 200
4) Mutasyon Oranı Örneğin 50
5) Çaprazlama Oranı Örneğin 55
6) Seçilen En iyi Örneğin 75

Uygulama Performans odaklı değildir görselliştirme odaklıdır. Yüksek değerlerde uygulamada crash olabilir.
